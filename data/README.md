# Dataset

This folder contains the dataset used in the study of territorial and temporal patterns of armed violence along the Colombia–Venezuela border.

## File

`colombia_venezuela_border_violence_panel_2011_2022.csv`

## Description

The dataset contains municipality-year observations for 222 municipalities located in the Colombia–Venezuela border region between 2011 and 2022.

Total observations: 2,664.

## Variables Included

- Victimization events
- Armed group presence (AGC, ELN, EPL, FARC, and FARC dissidents)
- Coca cultivation (hectares)
- Coca eradication (hectares)
- Laboratories destroyed
- Armed clashes
- Border municipality indicator
- Distance to the nearest border crossing
- Post-peace agreement indicator
- Presidential term

## Unit of Analysis

Municipality-year.

## Data Dictionary

A detailed description of all variables is available in:

`../docs/data_dictionary.csv`

## Notes

Department and municipality names are provided using their official names in Colombia.

Distances are measured in kilometers.

Binary variables are coded as:

- 1 = Presence / Yes
- 0 = Absence / No
